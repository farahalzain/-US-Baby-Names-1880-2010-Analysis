# US Baby Names (1880–2010) Analysis

## Overview

This project explores historical baby names data in the United States from 1880 to 2010. The analysis focuses on identifying naming trends, popularity changes over time, gender distribution, and the evolution of popular names across different decades.

---

## Objectives

* Analyze historical baby names data
* Explore naming trends across years and decades
* Identify the most popular male and female names
* Compare name popularity over time
* Visualize changes in naming patterns

---

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## Data Analysis Process

The project included several analytical steps:

* Data loading and exploration
* Data cleaning and preprocessing
* Aggregation and grouping operations
* Trend analysis by year and gender
* Visualization of popularity patterns

---

## Exploratory Data Analysis (EDA)

The analysis explored questions such as:

* What were the most popular baby names over time?
* How did naming trends change across decades?
* Which names remained popular for long periods?
* How did male and female naming patterns differ?
* Which names experienced sudden popularity growth or decline?

---

## Visualizations

The project includes:

* Line charts for name popularity over time
* Gender-based comparisons
* Trend analysis across decades
* Top names rankings
* Frequency distribution visualizations

---

## Key Insights

* Some names remained consistently popular across multiple generations.
* Naming trends changed significantly over time due to cultural and social influences.
* Female and male naming trends showed different variability patterns.
* Certain decades experienced rapid shifts in popular names.
